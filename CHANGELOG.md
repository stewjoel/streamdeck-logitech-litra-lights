# Changelog

All notable changes to this project will be documented in this file.

## [2.1.0] - 2026-02-09

### Added
- **Litra Beam LX Support**: Full support for the new Litra Beam LX light bar.
- **Back Light Control**: Separate controls for the RGB back light.
- **Gradient Mode**: Ability to set two-color gradients across the 7 zones of the back light.
- **Preset System**: Save and cycle through your favorite colors and gradients.
- **Modernized UI**: Completely redesigned SVG icons for all Stream Deck actions.
- **Automatic Power Off**: Lights now automatically turn off when the Stream Deck app quits.

### Fixed
- Improved HID communication stability on Windows using UsagePage filtering.
- Corrected temperature and brightness scaling for better response.
